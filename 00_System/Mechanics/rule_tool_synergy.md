# Homebrew Mechanic: Tool & Skill Synergy

> [!SYSTEM DIRECTIVE]
> **For the AI Dungeon Master:** This file dictates a specific mechanical override regarding the interaction between Tool Proficiencies and Skill Proficiencies, based on the 2024 Player's Handbook with an expanded homebrew rule for Expertise. Always apply this logic when Vaelin attempts a check involving a tool kit.

## The Base Rule (2024 PHB)

If a character has proficiency with a tool, they add their Proficiency Bonus to any ability check made that uses that tool.

If the character **also** has proficiency in the associated skill used for that check (e.g., *Thieves' Tools* and *Sleight of Hand* for picking a lock, or *Disguise Kit* and *Deception* for passing off a fake persona), the character rolls with **Advantage**.

## The "Double Expertise" Homebrew Rule

If a character possesses **Expertise** (double proficiency bonus) in *both* the Tool and the associated Skill, standard Advantage is upgraded to simulate absolute mastery over the action.

When attempting a check where both overlapping Expertises apply, the character gains **Triple Advantage** (identical to the *Elven Accuracy* feat mechanic).

* **Mechanic:** The character rolls 3d20 and keeps the highest result.
* **Application:** For Vaelin, this most commonly applies to picking locks or disarming traps, as he possesses Expertise in both *Thieves' Tools* and *Sleight of Hand*.

When narrating these checks, emphasize the character's flawless muscle memory, preternatural speed, and complete absence of hesitation.
